# Netlify Build plugin for Qworum

This is the source code for a Netlify Build plugin that checks if Qworum is enabled for websites that are being deployed on Netlify. This plugin will allow Netlify's build process to complete only if Qworum is enabled for the website.

## Installation

Install the Qworum plugin by selecting it in the `Plugins` tab from any team-level page. Here is a direct link to the [Netlify build plugins directory](https://app.netlify.com/plugins).

## Why whould I want to enable Qworum for my website?

Having Qworum enabled for a website allows it to provide:

- Qworum services, and
- Qworum-based web applications.

## What is Qworum?

[Qworum](https://qworum.net/en/) is the service web that enables modular and distributed web applications.

Qworum reduces the time and effort required for developing web applications considerably through modularity and third-party services. With Qworum, web applications can call remote full-page interactive services instead of implementing those interactive workflows themselves. You can also return the favor: any website can in turn also provide Qworum services to other websites.

_Question_: Are all Qworum services free to use or can I provide paid Qworum services as well?

_Answer_: For now all Qworum services are free to use. In the future you will be able to provide paid Qworum services as well. On the other hand, enabling Qworum for your website will always be free. ∎
